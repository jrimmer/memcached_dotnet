Implementation of memcached client for .NET framework
Based on Java client by Greg Whalin <greg@meetup.com>, http://www.whalin.com/memcached/

Maintained by Jason Rimmer <jrimmer@irth.net>

For more information on memcached see http://danga.com/memcached/

Has dependency on ICSharpCode.SharpZipLib for CRC and GZIP functionality
http://www.icsharpcode.net/OpenSource/SharpZipLib/
